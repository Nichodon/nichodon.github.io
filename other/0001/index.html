How to Embed Our Programs

19 July, 2017 – Nichodon

If you would like to embed a program into our website, use this syntax:

<iframe src="https://nichodon.github.io/programs/[id]/?embed=true"></iframe>

The [id] is a four-digit number (i.e. 0010). Note that you should set the width and height for the iframe.
