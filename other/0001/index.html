How to Embed Our Programs

19 July, 2017 – Nichodon

If you would like to embed a program into our website, use this syntax:

<iframe src="https://nichodon.github.io/programs/[id]/?embed=true"></iframe>

The [id] is a four-digit number (i.e. 0010). Note that you should set the width and height for the iframe. Ids of widgets can be found by hovering over them.

Disabled – contains a list of widget ids to be disabled, separated by commas.
Example: "?embed=true&disabled=meter1,meter2"

List of Tags Supported

Tag Name Main Value Disabled Description
input[type=number] Number ✓ Numberic text input
input[type=range] Number ✓ Slider
input[type=checkbox] "checked" or "unchecked" ✓ Checkbox
