Codon Coder

17 November, 2017 – Nichodon

Limitations: currently the Coder only recognizes one sequence at a time. For example, TACAAAACTTACTTTACT translates to Start Phe Stop.

How to use this thing: input into the DNA strand A, C, G, or T. The program will convert the DNA to RNA, and then the RNA to amino acids.
