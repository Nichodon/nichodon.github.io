Time Signature Metronome

21 June, 2017 – anematode

This is a customizable metronome. You can also input more complex time signatures into the numerator, using +'s to denote spaces between accents, commas to denote separate measures, and *'s to denote repetition. For example, 2*(3+2+3,) 4*2 will play two measures in the accent pattern X x x X x X x x and one measure in the accent pattern X x X x X x X x.
