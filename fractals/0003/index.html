Strange Attractors

31 May, 2017 – anematode

The Mandelbrot set; one of the most famous fractals (and, really, mathematical objects) of all time. It's striking, really, in how complex it is (pun not intended), yet how simple the rules are. Let's delve deeper into the Mandelbrot set and, hopefully, learn something new. If you're new to the ideas of fractals in the complex plane, you can see my intro here.

First, let's precisely define the Mandelbrot set. Define a function \(f_c(z)=z^2+c\). The Mandelbrot set is the set of all complex numbers \(c\) such that when the function is iterated, the resulting sequence's magnitude does not diverge to infinity.
